# Relation C Project

## Customer relationship management

### Technical

- ReactJS (Vite base)
- NestJs
- Postgresql
- RabbitMQ
- GRPC

### Local

- [Install NodeJs v18.0](https://nodejs.org/en)
- Start source code:

```bash
# Install dependencies
npm install

# Run source code
npm run start
```
